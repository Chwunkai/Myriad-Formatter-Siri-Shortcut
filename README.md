# Myriad-Formatter-Siri-Shortcut
A Siri Shortcut that converts number formatting from thousands separators (1,000) to myriad separators (1,0000), optimized for CJK numbering systems.
## Installation
Download the shortcut via [this iCloud link](https://www.icloud.com/shortcuts/fd52ca23e4d34892b799cded72fe734f) or from this repository, and allow clipboard access on first run.
## How to Use
You can trigger this shortcut in two ways:
1. **Manual Input**:
    - Open the **Shortcuts app** and tap the shortcut.
    - Enter the number you wish to convert in the pop-up prompt.
2. **Share Sheet**:
    - Highlight a number in any app (e.g., Safari, Notes).
    - Tap **Share** and select this shortcut from the list.
### Output
The converted number (e.g., 1,2345,6789) will be **automatically copied to your clipboard**.
